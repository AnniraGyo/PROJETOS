Apresentação

Esse site foi criado para mostrar as minhas habilidades como desenvolvedora front-end, e esse é o meu primeiro site criado com todos os meus estudos e anotações, para servir como portfólio inicial para futuros trabalhos. Algo sobre esse site é que ele ainda está em desenvolvimento, pois ainda preciso aprender como usar o JavaScript, mas quis deixar esse site como exemplo para medir minhas habilidades usando apenas HTML e CSS.
